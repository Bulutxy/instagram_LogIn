# Instagram Automation Bot

This project is an Instagram automation bot developed using Python and Selenium. It automates login, following, and unfollowing operations on Instagram.

## Features

* Automated Instagram login
* Follow users automatically
* Unfollow users automatically
* Session handling with Selenium WebDriver
* Simple and configurable script

## Technologies

* Python
* Selenium WebDriver

## Installation

1. Clone the repository:
   git clone https://github.com/Bulutxy/instagram_LogIn.git
   

3. Install dependencies:
   pip install selenium

4. Download and configure WebDriver (ChromeDriver or equivalent)

5. Update login credentials in the script

6. Run the script:
   python bot.py

## Usage

* Enter Instagram credentials in the configuration section
* Run the bot to automate follow/unfollow actions
